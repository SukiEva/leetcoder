# leetcoder

让本地调试变得更加优雅！

## 使用

1. 下载或 fork 本项目
2. 在项目根目录添加 input.txt，直接将 leetcode 样例放进去
3. 查看 Test 中的使用

## 功能

### 输入

具体在`src/util/Inputer.java`，可通过静态导入直接使用

- 整数数组，如`[1,2,3,4,5]`
- 字符串数组，如`["hello","world"]`
- 列表（同上）
- 二叉树，如`[4,1,6,0,2,5,7,null,null,null,3,null,null,null,8]`
- 链表，如`[1,2,3,4,5]`

### 输出

具体在`src/util/Printer.java`，可通过静态导入直接使用

- 树状二叉树
- 链表
- 各种数据结构
